# Building Sprite Assets

This folder contains 2D sprite images for village buildings.

## Required Sprites

Place PNG files (with transparency) for each building type:

- `castle.png` - Large medieval castle (recommended: 256x384px)
- `fortress.png` - Stone fortress/keep (recommended: 224x320px)  
- `farm.png` - Barn/farmhouse (recommended: 192x192px)
- `tower.png` - Tall watchtower (recommended: 96x448px)
- `temple.png` - Temple/church (recommended: 224x320px)

## Placeholder Generation

Currently using solid color fallbacks. Replace with actual sprite artwork for visual polish.

## Art Style Recommendations

- Pixel art or hand-drawn 2D
- Consistent lighting (front-facing)
- Transparent backgrounds (PNG alpha channel)
- High contrast for visibility
- Medieval/fantasy theme
